Layout

O layout é construido através de elementos semanticos, sendo eles:

  • <header> - Define o cabeçalho da pagina
  • <nav> - Define o menu de navegação da pagina
  • <section> - Define uma seção na pagina
  • <article> - Define um artigo na pagina
  • <aside> - Define um conteudo separado das sections (uma barra lateral)
  • <footer> - Define o rodapé na pagina
